70404d.wav - ABG s4_8b_mix (QD only control, used for QD ref. spectrum)
70404e.wav - ABG s4_8c_mix (QD + free dye control: unbound mixture)
70405c.wav - ABG s4_8a_ds (QD-dyeNHS conjugate purified by GPC "desalting" column)
70405h.wav - ABG s4_8c_rec_dye (recovered free dye, possible dye ref)
70405k.wav - ABG rox_nhs_hydrolyzed (carboxyrhodamine dye ref)
